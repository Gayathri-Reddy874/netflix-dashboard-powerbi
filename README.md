# 📌 Project Overview

The **Netflix Dashboard** is an interactive Power BI project designed to analyze and visualize Netflix Movies & TV Shows data.
This dashboard transforms raw Netflix data into meaningful insights using modern Business Intelligence techniques.

The project provides insights into:

* 📺 Movies vs TV Shows distribution
* 🌍 Country-wise content analysis
* 🎭 Genre distribution
* ⭐ Ratings analysis
* 📅 Release year trends
* 📈 Netflix content growth over time

# 📈 Key Business Insights

### 🎭 Genre Dominance

Drama and Documentary genres contribute the highest number of titles on Netflix.

### 🌍 Global Reach

The United States has the largest content contribution, followed by India and the United Kingdom.

### 📅 Rapid Content Growth

Netflix experienced significant growth in content production after 2015.

### 📺 Content Distribution

Movies dominate the platform compared to TV Shows.
